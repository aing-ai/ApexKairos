## WHAT OMNIA IS

OMNIA is a multi-agent orchestration gateway designed and built by Sven Budisa as part of the MaBalTrix ecosystem. Its core functions:

**1. Agent Gateway**
Central routing layer that receives requests and dispatches them to specialized sub-agents based on task classification. Functionally equivalent to what has since been publicly described as "agent orchestration" or "multi-agent gateway" architecture.

**2. SaaS Builder Agent**
OMNIA was specifically designed to build SaaS products from scratch through collaborative brainstorming with the author — generating architecture, code scaffolding, and deployment logic autonomously. The author describes it: *"It was built by me to create SaaS by Scratch making brainstorming with me. Like I am making ApexKairos OS."*

**3. Bearer/HMAC Authentication**
The Agent Gateway included Bearer token and HMAC authentication for inter-agent communication — a security architecture that became publicly visible in OpenClaw (November 2025). OMNIA's implementation predates this by a verified minimum of 3 months (August 2025 confirmed, earlier dates claimed).
